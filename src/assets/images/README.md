# images 폴더
## 폴더 구조
- common (Folder: common)
  > 공용적으로 사용할 이미지 저장
- foodproducts (Folder: foodproducts)
  > 음식 이미지 및 음식 폴더 내에서 공용으로 사용할 이미지 저장
- login (Folder: login)
  > 로그인에서 사용할 이미지 저장
- main (Folder: main)
  > 메인페이지에서 사용할 이미지 저장

## 이미지 파일 저장
자신이 작업하는 Component 폴더를 확인 한 후 그에 맞는 폴더에 이미지를 저장
> **※주의 : 이미지를 넣기 전 공용으로 사용 할 것인지 한번 더 생각 후 이미지 저장**
